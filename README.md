# IoT Ping Dashboard

Windows icin Flutter tabanli LAN ping izleme dashboard'u.

## Ozellikler

- Ardisik IP ping
- RTT / TTL goruntuleme
- Success / Fail sayac
- Status gostergesi
- Responsive desktop layout
- F10 ile guvenli cikis

## Kurulum

```powershell
flutter pub get
flutter run -d windows
flutter build windows
```

## Gereksinimler

- Flutter stable
- Windows Developer Mode (plugin icin)
